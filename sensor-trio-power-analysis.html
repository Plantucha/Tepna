How many windows to pin a sensor's σ?

The three-cornered hat recovers each device's HR-error σ from one simultaneous O2Ring + H10 + Verity window with no calibrated reference — but one window gives no confidence interval. This tool answers the practitioner's question: how many co-recorded windows do you need to pin each device's σ to a usable precision? Arm 1 — simulation power: a synthetic trio with known per-device σ (planted at the real estimates — O2Ring 2.72 / H10 1.86 / Verity 1.94 bpm, the raw-ECG 10-night broad hat) under a resting vs dynamic variance regime; the same TCH kernel is swept over N = 1…20 windows by Monte-Carlo → σ̂ bias, CI half-width and RMSE vs N. Arm 2 — anecdotal real comparison: the committed trio window(s) overlaid on the sim's predicted band as N accumulates (today N = 1, validation accumulating). A trio difference cancels the common true HR, so pure additive noise recovers with a regime-free 1/√N law — but at rest the shared beat-to-beat HRV is rendered differently by each device, injecting a signal-dependent correlated error that makes σ̂ noisier and can break the uncorrelated-error assumption. That is why the answer is "how many windows of what kind."

Minimum-N table — windows to reach a σ precision dynamic · CI half ≤ target

devicetrue σ (dyn)→ ±0.5→ ±0.25→ ±0.15
Read: the smallest N in {1,2,3,5,8,12,20} whose across-window 95% CI half-width falls at/below the target, in the clean (dynamic) regime where σ̂ is unbiased. One ~1-hour trio window already pins every device's σ to ±0.5 bpm; tightening to ±0.25 / ±0.15 costs several / many more. Because the three-cornered hat couples the trio, the noisiest corner sets a shared precision floor — so the absolute-± answer is roughly common across devices, not "the noisy device needs far more." Note that corner is now the O2Ring (σ 2.72 bpm), not the Verity (1.94): on the raw-ECG 10-night hat the optical wrist corner is the quietest of the three, reversing the ordering the earlier single-window estimate implied. The recommendation scales with the true σ ratio, so it generalizes beyond these planted values.
