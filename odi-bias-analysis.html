For OLS the slope's relative 95%-CI half-width is 1.96·√((1−R²)/(n−2))/√R² — it depends only on the correlation and n (the SD ratio cancels). Curve = that function at the assumed R²; knee (amber) = smallest n with CI within ±10% of the slope. The synthetic detector gives R²≈0.93 (a clean floor); real PSG is noisier, so set the slider to a literature-plausible value. “Bootstrap” resamples real-OxyDex synthetic points as a sanity check on the floor.
