How many nights?  Minimum reliable recording length per metric

A single night is one noisy draw of a person's true physiology. We measure ODI-4, rMSSD, and CGM-CV on every occasion of each stable (flat-arc) patient with the real detectors, decompose the variance into between-subject vs night-to-night, and apply the Spearman–Brown prophecy to read off the fewest nights/days each metric needs to be reliable.

Synthetic ground truth — read before citing. Subjects are flat-arc patients from the 1–12-night longitudinal lane (a stable latent per person), so within-subject variance is genuine night-to-night biology + detector noise, not a planted trend. Every occasion is scored by the real detector — ODI-4 (oxydex-dsp), rMSSD (pulsedex-dsp), CGM-CV (glucodex-dsp on the day-split continuous stream). Non-circular detector check (measured vs planted-latent ICC): — This certifies the reliability structure of the pipeline and recovers the planted variance ratio; it does not establish real-world clinical reliability, which needs a labelled repeated-measures cohort.
